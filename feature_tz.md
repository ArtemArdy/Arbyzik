# РАЗДЕЛ 1: PRODUCT VISION & METRICS

## 1.1 Проблема и решение

| Поле | Описание |
|------|----------|
| **Проблема** | Разработчики (Python, JS/TS, AI-инженеры) вынуждены переключаться между множеством AI-инструментов, вручную управлять контекстом и самостоятельно выбирать модели |
| **Решение** | Единый AI-хаб «Single UI Thread» — один интерфейс, автоматический выбор модели, персистентная память сессий |
| **Ключевая ценность** | Сложность скрыта под капотом. Пользователь разговаривает с одним ассистентом — система делает остальное |
